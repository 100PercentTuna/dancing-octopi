# Kunaal's Theme

A sophisticated WordPress theme for personal writing with gallery-grade editorial design. Minimalistic art-gallery aesthetic with strong typography and blue as the statement color.

**Version**: 4.2.0  
**Author**: Kunaal Wadhwa  
**Website**: [kunaalwadhwa.com](https://kunaalwadhwa.com)

---

## Features

### Custom Post Types
- **Essays**: Long-form articles with rich formatting
- **Jottings**: Quick, rough ideas and notes

### Block Library (43 Blocks)

#### Editorial Blocks (22)
- Insight Box, Pull Quote, Accordion, Sidenote
- Section Header, Takeaways, Citation, Aside
- Footnotes, Footnotes Section
- Magazine Figure, Lede Package
- Timeline, Glossary, Annotation
- Argument Map, Know/Don't Know
- Source Excerpt, Context Panel
- Related Reading

#### Analysis Blocks (11)
- Assumptions Register
- Confidence Meter
- Scenario Comparison
- Decision Log + Entry
- Framework Matrix (2x2/3x3)
- Causal Loop Diagram
- Evaluation Rubric + Row
- Debate (Steelman) + Side

#### Data Blocks (5)
- Publication Table
- Flow Chart + Step
- Chart (Bar, Line, Pie - vertical/horizontal)

#### Interactive Blocks (3)
- Parallax Section
- Scrollytelling + Step
- Reveal Wrapper

### Motion Primitives
- CSS animation utilities
- Scroll reveal classes
- Stagger utilities
- Respects `prefers-reduced-motion`

### PDF Export
- DOMPDF integration
- Journal-paper style layout
- Custom header/footer
- Color preservation

### Design System
- Typography-first approach
- Luxury-muted palette with blue accent
- Gallery minimalism
- Editorial sophistication

---

## Installation

1. Upload theme to `/wp-content/themes/kunaal-theme/`
2. Activate theme in WordPress admin
3. Install Composer dependencies: `composer install` (for PDF generation)
4. Configure Customizer settings

See `INSTALL.md` for detailed instructions.

---

## Requirements

- WordPress 6.0+
- PHP 8.0+
- Composer (for PDF generation)
